Utilizare Git

Pentru instalarea Git pe PC trebuiesc urmati pasii de instalare disponibili pe siteul oficial Git urmand acst Link


Pentru a utiliza GitHub trebuie urmati pasii de mai jos

1. creeaza un folder pentru depozitarea fisierului initial

2. In Visual Studio Code creeaza un fisier index.html pe care-l salvezi in folderul creeat

3. deschide comand prompt si mergi la folderul creeat urmand comanda

cd C:\ folder\folder.....

4. in comand prompt initializezi git, utilizand comanda

git init

5. apoi aduci fisierul creeat in status stages utilizand comanda

git add index.html

6. Si apoi facem commit pe acel fisier pentru a-l aduce in folderul git, utilizand comanda urmatoare

git commit -m "Initial Commit"

7. creem un repository in github

8. conectam folderul la repository si incarcam fisierul creeat urmand pasii din sectiunea

â€¦or push an existing repository from the command line

din portalul github


Alte comenzi git

Pentru update fisiere in github vom utiliza urmatorii pasi: in Visual Studio Code mergem la sectiunea Source Control apoi accesam meniul cu cele trei puncte, meniul Changes apoi Stage all changes

apoi mergem din nou la meniul cu trei puncte si selectam Commit all, in campul deschis in partea superioara trecem argumentarea commit-ului adica argumentarea modificarilor

Pentru a ridica aceste modificari pe GitHub, mergem din nou pe meniul cu trei puncte apoi selectam optiunea Push



Activare GitHub Pages

pentru ca paginile web stocate pe GitHub sa poata fi vizualizate online se va activa GitHub Pages urmand pasii de mai jos

Depozitul de pe GitHub trebuie sa fie un depozit public

Mergem apoi la setari

Apoi scroll down pana la sectiunea GitHub Pages iar de acolo la sectiunea Source selectam ramura proiectului


Corectie erori

1. daca apare eroarea

Remote origin already exists

se poate remedia utilizand codul urmator:

git remote set-url origin https://github.com/your/repository
